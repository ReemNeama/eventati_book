Returns all tasks with due dates within the specified date range

start The start date of the range (inclusive) end The end date of the range (inclusive)

The method includes a 1-day buffer on both ends to ensure tasks due exactly on the start or end date are included.

Implementation

List<Task> getTasksByDateRange(DateTime start, DateTime end) {
  return _tasks
      .where(
        (task) =>
            task.dueDate.isAfter(start.subtract(const Duration(days: 1))) &&
            task.dueDate.isBefore(end.add(const Duration(days: 1))),
      )
      .toList();
}
