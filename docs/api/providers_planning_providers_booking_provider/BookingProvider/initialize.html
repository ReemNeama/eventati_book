Future<void> initialize()

Initializes the provider by loading all bookings from SharedPreferences

This method should be called when the app starts, typically in main.dart or when the provider is first created. It loads all saved bookings from persistent storage and updates the provider's state. Notifies listeners when the operation completes.

Implementation

Future<void> initialize() async {
  _isLoading = true;
  _error = null;
  notifyListeners();

  try {
    // Load bookings from shared preferences
    final bookings = await _loadBookings();
    _bookings = bookings;

    _isLoading = false;
    notifyListeners();
  } catch (e) {
    _isLoading = false;
    _error = 'Failed to initialize bookings: $e';
    notifyListeners();
  }
}
