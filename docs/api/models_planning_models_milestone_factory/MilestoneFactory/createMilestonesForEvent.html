Create milestone instances for an event

Instantiates milestone objects for a specific event based on the appropriate templates for the event type. This method converts template milestones into actual milestone instances that can be tracked for a specific event.

Parameters:

  • eventId: The unique identifier of the event
  • eventType: The type of event ('wedding', 'business', 'celebration', etc.)

Returns: A list of Milestone objects that are instances (not templates) associated with the specified event. These instances can be tracked, completed, and rewarded.

Implementation

static List<Milestone> createMilestonesForEvent(
  String eventId,
  String eventType,
) {
  final templates = getTemplatesForEventType(eventType);

  return templates
      .map((template) => Milestone.fromTemplate(template, eventId))
      .toList();
}
