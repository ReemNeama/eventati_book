## Service Type-Specific Features

### Venue Services

- **Unique Features**:
  - Floor plans
  - Indoor/outdoor options
  - Amenities (parking, accessibility, etc.)
  - Capacity for different event types
  - Availability calendar

### Catering Services

- **Unique Features**:
  - Menu options (displayed in tabs)
  - Dietary accommodations
  - Staff options
  - Equipment included
  - Sample menus

### Photography Services

- **Unique Features**:
  - Portfolio samples
  - Equipment details
  - Team size options
  - Delivery timeline
  - Photography styles

### Planner Services

- **Unique Features**:
  - Specialization areas
  - Planning packages
  - Experience level
  - Services included
  - Planning approach

## Integration Points

- **Wizard Integration**: Services can be recommended based on wizard selections
- **Booking Integration**: Services connect to the booking system
- **Comparison Integration**: Services can be compared side-by-side
- **Planning Tools Integration**: Booked services appear in planning tools
